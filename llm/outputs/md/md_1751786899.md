# 西红柿炒鸡蛋食谱

## 食材准备
- 西红柿 2个（中等大小）
- 鸡蛋 3个
- 食用油 2汤匙
- 盐 1/2茶匙
- 白糖 1/2茶匙（可选）
- 葱花 少许（可选）

## 制作步骤

### 1. 食材处理
- 西红柿洗净后切块（可去皮）
- 鸡蛋打入碗中，加少许盐打散至均匀

### 2. 炒鸡蛋
1. 热锅凉油（1汤匙油）
2. 油热后倒入蛋液
3. 待底部凝固后用铲子划散
4. 炒至8成熟盛出备用

### 3. 炒西红柿
1. 锅中再加1汤匙油
2. 放入西红柿中火翻炒
3. 加盐和糖调味
4. 炒至西红柿出汤汁

### 4. 混合翻炒
1. 将炒好的鸡蛋倒回锅中
2. 与西红柿翻炒均匀
3. 撒上葱花（可选）
4. 翻炒10秒即可出锅

## 小贴士
- 喜欢汤汁多的可以加少许水
- 鸡蛋炒嫩些口感更好
- 可根据口味调整糖盐比例
